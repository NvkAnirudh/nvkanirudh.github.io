  • Some key learnings of the AWS Cloud Practitioner certification include understanding the fundamental concepts of cloud computing, including the benefits and drawbacks of cloud services, identifying the different types of cloud computing models, and understanding the AWS Global Infrastructure.

Google Data Analytics Professional Certificate

Summary

  • The Data Analytics Professional Certification offered by Google covers a range of topics related to data analytics, including collecting, transforming, analyzing, and visualizing data.
  • Some key learnings of the Google Data Analytics Professional certification include understanding how to use tools such as spreadsheets, SQL, and data visualization software to work with data. I learned about different types of data, such as structured and unstructured data, and how to clean, transform, and aggregate data to draw meaningful insights.
  • I also learned about statistical concepts such as hypothesis testing and regression analysis, and how to use these techniques to make data-driven decisions. Additionally, I learned about data privacy and ethics, as well as how to communicate data insights effectively.
  • Overall, the Google Data Analytics Professional certification provides a comprehensive understanding of the tools and techniques used in data analysis, making it a valuable asset for anyone interested in pursuing a career in data analytics.
