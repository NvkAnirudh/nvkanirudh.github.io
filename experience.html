Work Experience!

Research and Development Engineer in Machine Learning at IN-D in Intain Technologies Pvt Ltd

I worked on projects which involved Image processing, Computer Vision, Machine Learning, Neural Networks, and Python as the primary programming language. I was responsible for digitizing the Indian government documents using Optical Character Recognition (both Google Vision and PyTesseract) and extracting the required fields such as Name, Date of Birth, Address, et Cetra. I started with a rule-based engine where several rules were imposed on digitized text from OCR to extract information. This approach required knowledge of Python libraries such as NumPy and Pandas. I was responsible for processing three of the five documents the company processes. The overall accuracy of the extraction engine on the three documents crossed 90% comfortably. Later we extended the extraction logic to neural nets. We used Transfer Learning with Mask-RCNN for the object detection model. We train the model on required fields of around fifty samples with a similar format. As a result, the trained fields are detected when we pass a new document with the same format. The digitization is accurate using any OCR such as PyTesseract or cloud-based OCR upon these detected fields, with more than 95% in-turn accuracy.
